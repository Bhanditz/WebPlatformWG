## Specifications

W3C specifications use the following milestones:
* First Public Working Draft (FPWD)
* Editor's Draft (ED)
* Working Draft (WD)
* Candidate Recommendation (CR)
* Proposed Recommendation (PR)
* Recommendation (Rec)
